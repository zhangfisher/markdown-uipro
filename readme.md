# 表格

## 基本用法

完全兼容标准GFM表格

```markdown
<LiteTable>
| 表头 | 表头 | 表头 |
| ---- | ---- | ---- |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
</LiteTable>
```

## 合并单元格

完全兼容标准GFM表格

```markdown
<LiteTable>
| 表头 | 表头 | 表头 |
| ---- | ---- | ---- |
|{+1} R1C1 |  R1C3 |     // 合并R1C1-R1C2
| R1C1 | R1C2 | R1C3 |
| R2C1 | R2C2 | R2C3 |
|{+1+2} R3C1 | R3C2 | R3C3 | // 合并R3C1,R3C2,R4C1,R4C2
| R4C1 | R4C2 | R4C3 |
| R5C1 | R5C2 | R5C3 |

</LiteTable>
```

- 在`|`紧跟`{+<横向合并几格>+<纵向合并几格>}`
- 如果只需要纵向合并，则可以写`{+0+2}`
- `{}`只在紧跟`|`时有效，不能有空格

## 折分单元格

完全兼容标准GFM表格

```markdown
<LiteTable>
| 表头 | 表头 | 表头 |
| ---- | ---- | ---- |
|{-2} R1C1 |  R1C3 |     // R1C1拆分为两上单元格
|{-0-2} R1C1 | R1C2 | R1C3 | // 垂直拆分为两个单元格
| R2C1 | R2C2 | R2C3 |
|{+1+2-2} R3C1 | R3C2 | R3C3 | // 先合并R3C1,R3C2,R4C1,R4C2，再水平折分
| R4C1 | R4C2 | R4C3 |
| R5C1 | R5C2 | R5C3 |

</LiteTable>
```

- 在`|`紧跟`{-<水平拆分几格>-<垂直拆分几格>}`
- 如果只需要纵向合并，则可以写`{-0-2}`
- `{}`只在紧跟`|`时有效，不能有空格
- 允许同时使用合并和折分

## 多行文本

- 简单使用`\n`转义符

```markdown
<LiteTable>
| 表头 | 表头 | 表头 |
| ---- | ---- | ---- |
| R1C1 |  行1\n行2  |     
</LiteTable>
```

- 使用\`\`\`进行包裹

<LiteTable>
| 表头 | 表头 | 表头 |
| ---- | ---- | ---- |
|  |```
行1  
行2
行3  
行4
```| |
</LiteTable>

## 自定义样式

在`{,<样式字符串>}`为单元格添加自定义样式

```markdown
<LiteTable>
| 表头 | 表头 | 表头 |
| ---- | ---- | ---- |
| R1C1 |{,color:red;}  行1  |{}  行1|     
</LiteTable>
```

为整行添加`CSS样式`时，需要有第一个单元格添加

```markdown
<LiteTable>
| 表头 | 表头 | 表头 |
| ---- | ---- | ---- |
|{,>color:red;} R1C1 |  行1  |{}  行1|     
</LiteTable>
```

## 行标识

为表格行添加一个标识，以便使用突出行的样式，支持

- 支持`+`、`-`、`*`、`!`、`x`、`v`等标识。

```markdown
<LiteTable>
| 表头 | 表头 | 表头 |
| ---- | ---- | ---- |
|[+] R1C1 |  行1  |{}  行1|     // diff +
|[-] R1C1 |  行1  |{}  行1|     // diff -
|[*] R1C1 |  行1  |{}  行1|     // 重点
</LiteTable>
```

## 分割行

```markdown
<LiteTable>
| 表头 | 表头 | 表头 |
| ---- | ---- | ---- |
|  |  行1  |{}  行1|    
|-----------|     // 无标题
|  |  行1  |{}  行1|    
|----- 说明 ------|  // 具有标题
|:----- 说明 ------|  // 具有标题左对齐
|  |  行1  |{}  行1|    
</LiteTable>
```

## 展开折叠行

```markdown
<LiteTable>
| 表头 | 表头 | 表头 |
| ---- | ---- | ---- |
|  |  行1  |{}  行1|    
|(+3)  |  行1  |{}  行1|    // 以下三行可折叠与展开
|  |  行1  |{}  行1|    
|  |  行1  |{}  行1|    
|  |  行1  |{}  行1|    
</LiteTable>
```

## 样式引用

在单元格或行的样式可以直接引用预先声明的样式

- **样式变量:** 形如#样式名称=css样式
- **样式类变量:** 形如.样式名称=css样式
- **图标变量:** 形如图标名称=<data:image/svg+xml;base64,<...图标数据...>>

```markdown
<LiteTable>
.my=color:red;background:gray;

---

| 表头 | 表头        | 表头   |
| ---- | ----------- | ------ |
| R1C1 | {,@red} 行1 | {} 行1 |

</LiteTable>
```
